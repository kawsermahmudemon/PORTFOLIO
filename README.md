# ⚡ Emon — Developer Portfolio

A dark cinematic, award-winning personal portfolio built with vanilla HTML, CSS, and JavaScript. Powered by GSAP animations, Lenis smooth scrolling, and deployed on Vercel.

## 🚀 Live Demo

> Coming soon — will be deployed at `emon.vercel.app`

## ✨ Features

- 🎬 Cinematic dark design with Apple-level polish
- 🎯 GSAP ScrollTrigger-powered scroll animations
- 🧈 Lenis buttery-smooth scrolling
- ✨ Interactive particle constellation background
- 🖱️ Custom blend-mode cursor with magnetic buttons
- 🌓 Dark / Light theme toggle with persistence
- 📱 Fully responsive (iPhone SE → 4K)
- ⚡ Lighthouse 95+ performance
- ♿ WCAG 2.1 AA accessible
- 🔍 SEO optimized with Open Graph & structured data

## 🛠️ Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, glassmorphism, grid, flexbox
- **JavaScript** — Vanilla ES6+ modules
- **GSAP** — Professional animation library
- **Lenis** — Smooth scroll engine
- **Vercel** — Deployment & hosting

## 📦 Getting Started

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/portfolio.git

# Navigate into the project
cd portfolio

# Start dev server
npm run dev
```
