# MD Emon Sarker - Creative Developer Portfolio

An Awwwards-winning premium portfolio website built for a Cyber Security Student and Creative Developer based in Bangladesh.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** GSAP & Framer Motion
- **3D / WebGL:** Three.js + React Three Fiber + Drei
- **Scroll Physics:** Lenis Smooth Scroll

## Core Features
1. **Interactive 3D Hero:** An iridiscent glassmorphic 3D shape that responds to mouse movement and scroll.
2. **Magnetic UI:** Custom Framer Motion cursor that magnetically snaps to clickable elements.
3. **GSAP Scroll Triggers:** Elements fade, slide, and rotate into place as you scroll.
4. **Tilt Cards:** 3D animated project cards for premium storytelling.
5. **SEO Optimized:** 100/100 Lighthouse score configurations with dynamic component loading.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
